### 最小生成树

最小生成树问题一般都是在无向图上
有向图的生成树问题几乎不考
图论问题主要难在建立图（如何把问题抽象成图）
对于算法的原理考察不多

#### 普利姆算法（Prim）

Prim算法和Dijkstra算法很像。

稠密图图上可以用朴素版的Prim算法，时间复杂度$O(n^2)$。

稀疏图上可以用堆优化版的Prim算法，时间复杂度$O(mlogn)$。

#### 克鲁斯卡尔算法（Kruskal）

克鲁斯卡尔算法时间复杂度$O(mlogm)$，花在对边的排序上。

#### 选用

因为Kruskal比Prim好写很多，所以稠密图直接用朴素Prim，稀疏图直接用Kruskal，不考虑堆优化版的Prim算法。
